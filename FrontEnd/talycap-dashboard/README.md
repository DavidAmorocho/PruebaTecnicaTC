# Calificacion de TV shows

Este Proyecto fue generado con [Angular CLI](https://github.com/angular/angular-cli) version 17.3.17.

## Development server

Ejecute `ng serve` en la terminal. Ejecute la siguiente URL en google chrome `http://localhost:4200/`. La aplicacion se actualizara automaticamente cada vez que se realice un cambio dentro del codigo.

## Code scaffolding

Ejecute `ng generate component component-name` en la terminal para generar un nuevo componente.

## Build

Ejecute `ng build` para compilar el proyecto. Los artefactos de compilacion se almacenaran en el directorio `dist/`.

## Ejecucion de pruebas unitarias

Ejecute `ng test` para ejecutar las pruebas unitarias a traves de [Karma](https://karma-runner.github.io).


## Ayuda adicional

Para obtener ayuda para Angular CLI use `ng help` o consulte la pagina [Angular CLI Overview and Command Reference](https://angular.io/cli).
